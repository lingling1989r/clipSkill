# clipSkill — 用嘴剪音视频

> 把你说的话，变成剪辑指令。不用学软件，不用拖时间轴，说出来就剪好了。

---

## 它能做什么

你录了一段播客、访谈、课程、直播回放——现在你想剪掉废话、提取精华、输出成品。

以前你要：打开剪辑软件 → 听一遍 → 找时间点 → 拖轴 → 导出。

现在你只需要在 bot 里说一句话：

```
把前面讲吃饭的那段去掉
```

clipSkill 会自动把音频上传、转录成文字、找到对应片段、剪掉它，然后告诉你结果在哪里听。

## 工作原理

```
你的音视频文件
      ↓
  上传到 PodAha
      ↓
  AI 转录成文字（Whisper）
      ↓
  AI 理解你的指令，定位片段
      ↓
  自动剪辑、拼接
      ↓
  结果发布到你的账号
      ↓
  直接在 podaha.com 收听 / 下载
```

所有结果都绑定到你的 PodAha 账号，随时可以在网页上回听、继续编辑。

## 快速开始

### 第一步：获取 Token

1. 注册登录 **[podaha.com](https://podaha.com)**（免费）
2. 进入 **Settings → API Tokens**
3. 点击「创建 Token」，复制（格式为 `pat_` 开头）

### 第二步：配置

```bash
cp .env.example .env
```

编辑 `.env`：

```
CLIP_SKILL_TOKEN=pat_你的token
```

### 第三步：验证连接

```bash
node scripts/smoke.mjs
```

看到 `All checks passed ✓` 就可以用了。

### 第四步：开剪

```bash
node scripts/run.mjs --file 你的访谈.mp3
```

完成后会输出：

```
[clip-skill] ✓ DONE
[clip-skill] view on platform: https://podaha.com/audio/xxx
```

打开链接，结果在里面。

## 常见问题

**Q：支持多长的音频？**
A：本地模式无限制，云端模式建议单文件不超过 2 小时。

**Q：转录准确率如何？**
A：中文使用 Whisper + 阿里云 ASR，普通话准确率 95% 以上。方言、口音较重时建议指定 `--language zh`。

**Q：剪辑结果能下载吗？**
A：可以，登录 podaha.com 后在音频详情页直接下载 MP3。

**Q：我的音频数据安全吗？**
A：文件上传到你自己账号下，仅你可见，不会用于训练。
